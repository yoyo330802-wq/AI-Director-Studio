## 二、系统架构设计

### 2.1 整体架构

```
CDN层 (CloudFlare/七牛云)
        ↓ HTTPS
前端层 (Next.js 14)
  /studio - 创作工作台
  /gallery - 作品广场
  /templates - 提示词模板库
  /pricing - 套餐购买
  /dashboard - 用户中心
        ↓ REST API / WebSocket
网关层 (Kong API Gateway)
  认证插件 (JWT/OAuth) | 限流插件 (令牌桶) | 缓存插件 (Redis)
        ↓
应用层 (FastAPI 0.110+)
  用户服务 | 智能路由引擎 | 计费服务 | 任务队列 | 回调处理 | 管理后台
        ↓
上游渠道
  ┌─────────────┬─────────────┬─────────────┐
  │ 自建Wan2.1  │ 硅基流动     │ Vidu        │
  │ 主力50%     │ 主力30%     │ 主力20%     │
  │ 4×H100     │ 多模型      │ 动漫优化    │
  │ 0.025元/秒  │ 动态路由    │ 0.05元/秒   │
  └─────────────┴─────────────┴─────────────┘
        ↓
数据层
  PostgreSQL (用户/订单) | MongoDB (作品元数据) | Redis (缓存/队列) | MinIO/OSS (视频存储)
```

### 2.2 智能路由核心算法

支持多维度智能路由：成本优先、质量优先、智能平衡三种模式

**路由决策优先级：**
1. 强制指定（企业用户）
2. 质量优先模式 → 可灵
3. 成本优先模式 → Wan2.1-1.3B
4. 智能平衡模式（默认）

### 2.3 任务队列

使用 Celery + Redis 实现异步任务处理，支持任务回调和重试机制
